# Happy Plant
This project uses BLE GATT to transmit soil moisture sensor data from ESP32 microcontrollers to an iPhone app called Happy Plant. Each ESP32 reads an analog soil moisture sensor, calibrates the raw value to a percentage, and exposes it via a custom GATT service as a readable/notifiable characteristic. The firmware is written using the Arduino IDE with the ESP32 BLE library. An iPhone app built with Swift/SwiftUI uses CoreBluetooth to scan for, connect to, and read from these peripherals, displaying the most recent moisture level and time since last update for each device.
